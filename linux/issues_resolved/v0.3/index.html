Issue Resolution

Complete breakdown of the Linux .deb crash and its resolution

The Problem

v0.2
$ pyllamaui
Traceback (most recent call last):
  File "/usr/bin/pyllamaui", line 6, in <module>
    import markdown2
ModuleNotFoundError: No module named 'markdown2'

The v0.2 Linux .deb package failed to include the python3-markdown2 dependency in its control file, causing the application to crash immediately after installation.

Root Cause

The dependency was not declared in the Debian package control file, causing fresh installations to fail on startup.

The Fix (v0.3)

  • Added python3-markdown2
  • Rebuilt .deb installer
  • Verified launch stability
