System Overview

Privacy First

No data collection. No telemetry. Zero internet dependency after initial setup. Your conversations never leave your machine.

Lightweight Core

Built on Python 3.10+ with customtkinter frontend. Minimal resource overhead designed for efficient local inference.

Developer Native

Available as standalone desktop app, VS Code extension, or Python package. Integrates seamlessly into your workflow.

Architecture Philosophy

PyLlamaUI serves as a bridge between powerful local models and an accessible interface, eliminating the need for command line interaction or cloud-based services. The core philosophy is pure on-device AI.

Features & Capabilities

Interactive Chat UI

Clean, responsive interface available as standalone desktop app or directly inside VS Code. Distraction-free workspace design.

Offline Markdown Rendering

Real-time formatting for bold text, headers, lists, blockquotes, and tables. No internet required for rendering.

Code Capabilities

Syntax highlighting for Python, JavaScript, JSON, and Bash. One-click copy buttons preserve proper indentation.

Model Flexibility

Native support for any Ollama-compatible model: TinyLLaMA, Gemma, DeepSeek-Coder, Llama 3, and more.

example.py
# Pull and run any Ollama model
$ ollama run deepseek-coder
# Python integration example
import pyllamaui
 
# Initialize client
client = pyllamaui.Client(
    model="llama3",
    offline=True
)
 
# Stream response
for chunk in client.chat(
    "Explain quantum computing"
):
    print(chunk, end="")

quickstart.sh
#!/bin/bash
# PyLlamaUI Quick Installation
 
echo "Initializing PyLlamaUI setup..."
 
# 1. Ensure Ollama is running
if ! command -v ollama &> /dev/null; then
    echo "Error: Ollama not found. Please install first."
    exit 1
fi
 
# 2. Install PyLlamaUI
pip install --upgrade pyllamaui
 
# 3. Launch
pyllamaui --launch
